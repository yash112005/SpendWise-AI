# SpendWise AI

Modern AI spend auditing platform for startups and developers.

SpendWise AI helps users analyze spending on AI tools like Cursor, Claude, ChatGPT, GitHub Copilot, Gemini, and APIs to identify overspending and optimize subscriptions.
